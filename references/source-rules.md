# Source Rules

## 示例股票池

- 公共版默认股票池放在 `references/default-stocks.json`
- 这是**示例配置**，不是必须使用的正式股票池
- 开源发布前可以保留示例，也可以替换成自己的公开样例

示例条目格式：

```json
[
  {"name": "招商银行", "code": "600036", "prefix": "sh"},
  {"name": "中信建投证券", "code": "06066", "prefix": "hk"}
]
```

## 数据源

### 1. 分红明细（正式表格主 source）
- A 股页面：同花顺 F10 `https://basic.10jqka.com.cn/{code}/bonus.html`
- 港股页面：同花顺 F10 `https://basic.10jqka.com.cn/HK{code}/bonus.html`
- 用途：抓取逐笔分红、特别分红、补充分红
- 原因：正式表格不能只依赖单一 API；逐笔分红核对时需要更完整的来源

### 2. 当前价格
- API：腾讯财经 `https://qt.gtimg.cn/q={prefix}{code}`
- A 股示例：`sh600036`
- 港股示例：`hk06066`
- 用途：分析报告现价、2025/2026 股息率分母

### 3. 历史价格
- A 股：Baostock
  - 用途 A：分析报告均线（前复权）
    - 日线/周线/月线均线：`adjustflag="2"`
  - 用途 B：2024 年逐笔分红股息率分母（除息日收盘价）
    - 除息日收盘价：`adjustflag="3"`
- 港股：东方财富历史 K 线 API
  - 用途 A：分析报告均线（前复权）
    - 日线：`klt=101, fqt=1`
    - 周线：`klt=102, fqt=1`
    - 月线：`klt=103, fqt=1`
  - 用途 B：2024 年逐笔分红股息率分母（除净日收盘价）
    - 日线：`klt=101, fqt=0`

## 分红口径

### 年度归属
- 按 **除权除息日** 所在年份归属
- 不按 `REPORT_DATE` 归属

### 股息率公式
- `2026至今股息率` = 2026年至今已除息分红累计 ÷ 10 ÷ 当前价
- `2025全年股息率` = 2025年内已除息分红累计 ÷ 10 ÷ 当前价
- `2024全年股息率` = 2024年每次分红分别 ÷ 当次除息日收盘价，再把单次股息率相加

## 分析报告章节 2 规则

### 2.1 250日价格/低
- 逻辑：`当前价格 <= 250日均线 * 1.0027`
- 展示：偏差率 = `(当前价格 - 250日均线) / 250日均线`

### 2.2 120日在60日低
- 逻辑：`|MA120 - MA60| / MA60 <= 0.0027`
- 展示：偏差率

### 2.3 30周在20周附近
- 逻辑：`|MA30W - MA20W| / MA20W <= 0.0027`
- 展示：偏差率

### 2.4 当前股价在60周线附近或者以下
- 逻辑：`当前价格 <= 60周均线 * 1.0027`
- 展示：偏差率 = `(当前价格 - 60周均线) / 60周均线`

### 2.5 20周线是不是在60周线附近及以下
- 逻辑：`MA20W <= MA60W * 1.0027`
- 展示：偏差率 = `(MA20W - MA60W) / MA60W`
- 备注：满足条件记为 **小周期休息**；超过 `1.0027` 倍记为 **小周期干活**

### 2.6 20月线是不是在60月线附近及以下
- 逻辑：`MA20M <= MA60M * 1.0027`
- 展示：偏差率 = `(MA20M - MA60M) / MA60M`
- 备注：满足条件记为 **大周期休息**；超过 `1.0027` 倍记为 **大周期干活**

## 章节 1 方向箭头

- 当前价格后的 3 个箭头，必须来自**最近 4 个已完成交易日收盘价**推导出的 3 段方向
- 不要把实时 `current price` 混进箭头计算

## 输出文件

### 分红总表
- 文件名：`股票分红信息整合总表_YYYY-MM-DD.md`
- 默认目录：skill 本地 `output/`
- 可通过 `--out-dir` 覆盖

### 分析报告
- 文件名：`股票分析表格_版本11_YYYY-MM-DD.md`
- 默认目录：skill 本地 `output/`
- 可通过 `--out-dir` 覆盖

## 开源发布前检查

- 删除或替换任何私人 NAS / 云盘路径
- 删除或替换任何 Feishu / Telegram / Discord / Slack 群组或用户 ID
- 删除任何私有文档链接、飞书文件夹 token、message_id、open_id
- 删除个性化口头禅、署名、宠物设定、组织内流程
- 确保股票池来自公开样例文件，而不是私有持仓或私人监控名单
- 如果文档里仍有“同步到某个私有云盘 / 文档镜像”的步骤，改成可选说明或完全移除
